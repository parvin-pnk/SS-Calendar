## How to Run this

put the follwing in the Terminal
- npx create-react-app my-calendar-app
- cd my-calendar-app
- npm install
- npm install dayjs
- npm install -D tailwindcss@npm:@tailwindcss/postcss7-compat postcss autoprefixernpx tailwindcss-cli init -p
